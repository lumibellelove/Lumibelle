Guide

음원 재생, 마이크 상태, 리허설 사운드 체크를 담당합니다.

음향 담당 가이드

  • 리허설 전 음원 파일명과 재생 순서 확인
  • 팀별 무선 마이크 수량 확인
  • SE / 곡 / MC 구간 재생 기준 확인
  • 곡 제목을 말하면 재생하는 팀은 큐시트에 표시
  • MC 구간 BGM 반복 재생 여부 확인
  • 음원 교체나 재생 오류 발생 시 총괄에게 즉시 보고
  • 공연 종료 후 음원/장비 반납 상태 확인

보고 기준

돌발 상황, 지연, 관객 민원, 장비 문제는 즉시 총괄 또는 부총괄에게 보고해 주세요. 내부 링크와 운영 내용은 외부 공유 금지입니다.
