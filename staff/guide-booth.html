물판/특전회 스탭은 루미 ID, 오늘 번호, 메아테, 포인트, 특전권 사용, 타이머를 처리합니다.

가이드 선택으로 특전회 체크 열기 스탭 허브로

특전회 기본 흐름

1. 팬 조회 / 신규 발급루미 ID 또는 닉네임으로 조회하고, 없으면 스탭 확인 후 신규 발급합니다.
2. 오늘 번호 확인오늘 번호는 당일 특전회에서 빠르게 찾기 위한 임시 번호입니다.
3. 특전회 참가 처리실제 촬영/교류를 진행할 팬만 특전회 처리로 넘어갑니다.
4. 권종 → 멤버 → 1장 저장일반 특전권, Welcome Ticket, Join Ticket 등 권종을 선택하고 멤버별 1장씩 저장합니다.
5. 현재 팬 유지 / 다음 팬같은 팬이 풀루프면 유지, 다른 팬이면 다음 팬으로 넘어갑니다.

오늘 이벤트 시작 / 오늘 번호 기준

오늘 번호오늘 번호는 당일 현장용 임시 번호입니다. 루미 ID와 다르게 날짜/이벤트마다 새로 부여됩니다.
운영 시작 전책임자 또는 메인 스탭이 [오늘 시작] 버튼을 눌러 오늘 번호를 001부터 시작합니다.
초기화 내용[오늘 시작]을 누르면 오늘 번호, 최근 팬 목록, 오늘 처리 기록, 물판 처리 기록, 주의 기록, 현재 처리 중 팬, 타이머에 연결된 팬 정보가 초기화됩니다.
운영 중 주의운영이 이미 시작된 뒤에는 임의로 누르지 않습니다. 초기화가 필요한 경우 반드시 책임자 확인 후 진행합니다.

지급 / 적립 기준표

물판/특전회 스탭은 아래 기준대로만 지급합니다. 헷갈리면 책임자에게 확인해 주세요.

메아테 지정메아테 특전권 1장 지급 + 물판 포인트 1P 적립 + 루미 방문 기록
주최 라이브 메아테메아테 특전권 1장 지급 + 물판 포인트 2P 적립 + 루미 방문 기록
신규 이벤트 참가Welcome Ticket 1장 지급. 메아테 특전권과 별도 사유로 기록

특전권 15장 구매물판 포인트 1P 적립

지급 사고 방지

메아테 특전권과 Welcome Ticket 구분메아테 특전권은 메아테 지명자용, Welcome Ticket은 신규 이벤트 참가자용입니다.
중복 수령 확인이벤트권과 특전권은 루미 ID 기준으로 지급/사용 기록을 남깁니다.
모르면 지급 보류조건이 애매하면 즉시 지급하지 말고 책임자에게 확인합니다.

타이머 기준

멤버별 타이머마리링 / 루루 / 이로 / 루나 각각 독립적으로 작동합니다.
마무리 알림60초는 25초 전, 90초는 35초 전, 120초는 50초 전, 180초는 60초 전에 안내합니다.
종료 안내시간 종료 시 “○○님, 시간 되셔서 슬슬 마무리 부탁드릴게요~”라고 안내합니다.
연장 기준대기 팬 없음 + 멤버 부담 없음 + 진행 지장 없음일 때만 30~60초 연장 가능합니다.

주의 상황

재확인 거부기록 누락 방지와 멤버별 정산 정확도를 위한 공통 기준이라고 안내합니다.
멤버 불편대기 팬이 없어도 연장하지 않고 마무리 안내를 진행합니다.
정정/환불/분쟁스탭 단독 판단하지 말고 책임자 확인 후 처리합니다.
