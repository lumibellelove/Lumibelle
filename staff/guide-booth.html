물판/특전회 스탭은 루미 ID, 오늘 번호, 메아테, 포인트, 특전권 사용, 타이머를 처리합니다.

가이드 선택으로 특전회 체크 열기 스탭 허브로 멤버 허브 열기

특전회 기본 흐름

1. 팬 조회 / 신규 발급루미 ID 또는 닉네임으로 조회하고, 없으면 스탭 확인 후 신규 발급합니다.
2. 오늘 번호 확인오늘 번호는 당일 특전회에서 빠르게 찾기 위한 임시 번호입니다.
3. 특전회 참가 처리실제 촬영/교류를 진행할 팬만 특전회 처리로 넘어갑니다.
4. 권종 → 멤버 → 1장 저장일반 특전권, Welcome Ticket, Join Ticket 등 권종을 선택하고 멤버별 1장씩 저장합니다.
5. 현재 팬 유지 / 다음 팬같은 팬이 풀루프면 유지, 다른 팬이면 다음 팬으로 넘어갑니다.

오늘 이벤트 시작 / 오늘 번호 기준

오늘 번호오늘 번호는 당일 현장용 임시 번호입니다. 루미 ID와 다르게 날짜/이벤트마다 새로 부여됩니다.
운영 시작 전책임자 또는 메인 스탭이 [오늘 시작] 버튼을 눌러 오늘 번호를 001부터 시작합니다.
초기화 내용[오늘 시작]을 누르면 오늘 번호, 최근 팬 목록, 오늘 처리 기록, 물판 처리 기록, 주의 기록, 현재 처리 중 팬, 타이머에 연결된 팬 정보가 초기화됩니다.
운영 중 주의운영이 이미 시작된 뒤에는 임의로 누르지 않습니다. 초기화가 필요한 경우 반드시 책임자 확인 후 진행합니다.

지급 사고 방지

메아테 특전권과 Welcome Ticket 구분메아테 특전권은 메아테 지명자용, Welcome Ticket은 신규 이벤트 참가자용입니다.
중복 수령 확인이벤트권과 특전권은 루미 ID 기준으로 지급/사용 기록을 남깁니다.
모르면 지급 보류조건이 애매하면 즉시 지급하지 말고 책임자에게 확인합니다.

타이머 기준

멤버별 타이머마리링 / 루루 / 이로 / 루나 각각 독립적으로 작동합니다.
마무리 알림60초는 25초 전, 90초는 35초 전, 120초는 50초 전, 180초는 60초 전에 안내합니다.
종료 안내시간 종료 시 “○○님, 시간 되셔서 슬슬 마무리 부탁드릴게요~”라고 안내합니다.
연장 기준대기 팬 없음 + 멤버 부담 없음 + 진행 지장 없음일 때만 30~60초 연장 가능합니다.

주의 상황

재확인 거부기록 누락 방지와 멤버별 정산 정확도를 위한 공통 기준이라고 안내합니다.
멤버 불편대기 팬이 없어도 연장하지 않고 마무리 안내를 진행합니다.
정정/환불/분쟁스탭 단독 판단하지 말고 책임자 확인 후 처리합니다.

신규팬 / 유령 방지 안내

줄이 비는 시간은 멤버 멘탈에 직접 영향을 줄 수 있습니다. 팬이 망설이지 않도록 스탭이 먼저 부드럽게 안내해주세요.

처음 온 팬“처음 오셨어요? 괜찮아요, 천천히 안내해드릴게요. 특전권 확인 후 원하는 멤버 줄로 가시면 됩니다.”
빈 줄 안내“지금 바로 촬영 가능하세요. 처음이신 분도 포즈 추천해드릴 수 있어요.”
오시 고민“오시 고민 중이면 편하게 찍고 싶은 멤버와 먼저 찍어보셔도 괜찮아요.”
특전권 미준비“촬영 전 특전권 먼저 준비해주시면 바로 안내해드릴게요.”

접촉 / 밀착 / 불편 포즈 대응

즉시 차단멤버 신체, 의상, 소품 접촉 / 접촉처럼 보이는 포즈 / 과도한 밀착 / 신체 부위 관련 발언은 진행하지 않습니다.
1차 멘트“해당 포즈는 멤버 보호 규정상 진행이 어렵습니다. 다른 포즈로 부탁드립니다.”
반복 시“같은 요청이 반복될 경우 특전 진행이 제한될 수 있습니다. 규정 내 포즈로 진행해주세요.”
실제 접촉 발생특전 진행을 중단하고 책임자에게 즉시 공유합니다. 필요 시 당일 이후 참여 제한을 검토합니다.

리프 / 멘부리 / 타계 발언 대응

리프 요구“리프와 마음은 멤버 컨디션과 일정에 따라 자율적으로 이루어지며, 모든 글에 답변을 보장하지 않습니다.”
타계 발언멤버가 붙잡거나 감정적으로 달래지 않게 합니다. 반복되면 책임자에게 공유합니다.
멘부리 장문멤버가 상담자 역할을 하지 않게 하고, 공식 문의 또는 책임자 확인으로 넘깁니다.
타팬 비교비교성 발언은 멤버에게 전달하지 않고, 필요 시 “모든 팬에게 동일한 답변을 보장하지 않습니다”로 안내합니다.

숙제체키 접수 / 수령 기준

숙제체키는 기억이 아니라 번호와 기록으로 관리합니다. 팬 문의를 줄이기 위해 접수 시점부터 번호를 남겨주세요.

번호 생성예: HW-0712-001. 체키 뒷면에도 “번호 / 루미 ID / 멤버명”을 작게 남깁니다.
접수 기록팬 닉네임, 루미 ID, 촬영 멤버, 요청 문구, 사진 식별 메모, 수령 예정 행사, 수령 방법을 기록합니다.
멤버 전달멤버 전용 허브의 숙제체키 목록으로 확인하게 하고, 완료 후 완료 체크를 하도록 안내합니다.
팬 상태 표시루미폰에는 접수됨 / 준비 중 / 수령 가능 / 수령 완료처럼 단순한 상태로 보여주는 것을 목표로 합니다.
수령 완료현장에서 전달 후 반드시 수령 완료 체크를 남깁니다. 장기 미수령은 별도 목록으로 관리합니다.

멤버 SOS 발생 시

멤버 신호멤버가 “스탭님 확인 부탁드려요”라고 말하면 즉시 개입합니다.
팬에게 안내“잠시 확인하겠습니다”라고 말한 뒤 멤버가 직접 설명하지 않게 합니다.
상황 기록접촉, 사생활 질문, 리프 압박, 타팬 비교, 스탭 지시 불응은 스탭 메모톡 또는 책임자에게 공유합니다.
멤버 컨디션멤버가 불편함을 호소하면 대기 팬이 없어도 연장하지 않고 마무리합니다.
