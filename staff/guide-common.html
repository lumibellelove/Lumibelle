모든 스탭이 먼저 확인해야 하는 공통 운영 기준입니다.
현장에서는 친절하게, 모르는 건 책임자에게 확인해 주세요.

가이드 선택으로 스탭 허브로 멤버 허브 열기

스탭 기본 태도

정중하게 안내팬에게 먼저 차분하고 부드럽게 안내합니다.
모르면 책임자 확인예외 상황, 환불/정정, 민원은 혼자 단정하지 않습니다.
내부 내용 외부 발설 금지멤버 개인 사정, 운영 내부 기준, 시트 구조는 외부에 말하지 않습니다.
문제 상황 기록항의, 안내 불응, 멤버 불편, 타 팬 민원은 주의 체크 또는 책임자에게 공유합니다.

루미 ID 기본 원칙

자동 발급하지 않음팬이 원하거나 루미벨 관련 혜택/기록이 필요한 경우에만 발급합니다.
QR은 입구QR은 루미 ID 조회 페이지로 이동하는 용도이며, PIN 4자리는 QR에 넣지 않습니다.
사전 발급 / 현장 발급팬이 직접 만들거나, 현장에서 스탭이 보조 발급할 수 있습니다.

기록 구분

입장 확인공연 입장 처리입니다. 루미 체크인이나 스탬프가 아닙니다.
루미 방문 기록루미벨 관련 접점이 확인된 방문 기록입니다.
루미 체크인촬영·교류 특전에 실제 참여한 기록입니다.
스탬프루미 체크인 완료 시 지급되는 참여 보상성 기록입니다.
물판 포인트특전권 15장 구매 / 메아테 지정 / 주최 라이브 메아테 지정 기준입니다.
XP·홈페이지 포인트물판 포인트와 별도 시스템입니다.

개인정보 / 보안

시트 원본 노출 금지시트 원본, 예약 정보 전체, 팬 개인정보 화면을 팬에게 보여주지 않습니다.
필요한 정보만 확인예약번호, 입금자명, 이메일 등은 확인이 필요한 경우에만 사용합니다.
캡처/공유 금지팬 정보와 내부 기록을 개인적으로 캡처하거나 공유하지 않습니다.

멤버 SOS를 받는 기준

멤버가 “스탭님 확인 부탁드려요”라고 말하면, 즉시 멤버 대신 상황을 받아주세요. 멤버가 팬과 직접 실랑이하지 않게 하는 것이 우선입니다.

바로 개입접촉·밀착 포즈, 사생활 질문, 답변 요구, 성적/불쾌 발언, 멤버가 불편해하는 상황은 바로 스탭이 받습니다.
멤버를 뒤로 빼기필요하면 “잠시 확인하겠습니다”라고 안내하고 멤버가 물을 마시거나 정리할 시간을 줍니다.
기록 남기기반복 가능성이 있는 상황은 스탭 메모톡 또는 책임자에게 남깁니다. 감정 평가가 아니라 재발 방지용 기록입니다.

현장 문제 대응 멘트

불편 포즈 요청“해당 포즈는 멤버 보호 규정상 진행이 어렵습니다. 다른 포즈로 부탁드립니다.”
반복 요청“같은 요청이 반복될 경우 특전 진행이 제한될 수 있습니다. 규정 내 포즈로 진행해주세요.”
리프/맘찍 문의“리프와 마음은 멤버 컨디션과 일정에 따라 자율적으로 이루어지며, 모든 글에 답변을 보장하지 않습니다.”
사생활 질문“멤버 개인 사정이나 사생활 관련 질문은 답변드리기 어렵습니다. 루미벨 활동 관련 문의는 공식 문의폼을 이용해주세요.”
공식 제보 안내“운영 관련 문의 및 제보는 공식 문의폼으로 접수해주세요. 날짜, 상황, 관련 자료를 함께 남겨주시면 확인하겠습니다.”

주의 메모 체크 기준

소통 압박리프/맘찍 반복 요구, 답변 여부로 멤버 압박, 타팬 비교, 타계·탈덕 발언으로 멤버를 흔드는 경우.
안전 문제접촉/밀착 포즈 요구, 실제 접촉 또는 접촉 시도, 성적 발언, 신체 부위 언급, 음주 상태, 스탭 지시 불응.
사생활 문제DM/사적 연락 요구, 퇴근길·동선 질문, 본업·학교·거주지 등 개인 정보 질문 반복.
처리 상태단순 메모 / 구두 안내 완료 / 경고 완료 / 당일 특전 제한 / 책임자 확인 필요 / 향후 참여 제한 검토 중 하나로 남깁니다.

익명 커뮤니티 / 공식 문의 기준

공식 접수 기준익명 커뮤니티 게시글은 공식 접수로 간주하지 않습니다. 확인이 필요한 사안은 공식 문의폼으로 안내합니다.
멤버 보호멤버에게 익명 글 확인을 요구하지 않습니다. 필요한 내용은 운영진이 공식 문의와 현장 기록 기준으로 확인합니다.
자료가 있는 제보날짜, 장소, 상황, 관련 자료가 있는 제보만 책임자가 검토합니다. 감정적 조리돌림은 운영 기준으로 삼지 않습니다.
