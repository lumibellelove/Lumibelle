Guide

스크린 영상 송출, 대기 영상, 팀별 영상, 이벤트 이미지를 담당합니다.

영상 담당 가이드

  • 리허설 전 스크린 테스트 및 파일 재생 확인
  • 팀별 영상 파일명과 순서 확인
  • 음원과 영상 싱크가 필요한 경우 합본 파일 우선 사용
  • 대기 영상 / 로고 영상 / 이벤트 이미지 송출 준비
  • 본 공연 중 큐시트 기준으로 영상 송출
  • 영상 임무 종료 시 총괄 확인 후 퇴근 가능
  • 오류 발생 시 화면을 대기 이미지로 전환 후 즉시 보고

보고 기준

돌발 상황, 지연, 관객 민원, 장비 문제는 즉시 총괄 또는 부총괄에게 보고해 주세요. 내부 링크와 운영 내용은 외부 공유 금지입니다.
