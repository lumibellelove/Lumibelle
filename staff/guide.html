Shine Me UP 기본 정보

공연 정보 2026년 7월 12일 (일) / KT&G 상상마당 홍대 라이브홀
실무 책임 총괄: 페리 / 부총괄: 연
운영 원칙 스탭 단독 판단이 애매한 상황은 바로 총괄 또는 부총괄에게 공유합니다.

긴급 호출 기준

총괄 호출
  • 대관처 / 외부 관계자 응대
  • 큰 클레임 또는 안전 문제
  • 공연 시간 변경 / 진행 변경
  • 멤버 관련 긴급 상황
  • 최종 의사결정이 필요한 상황
부총괄 호출
  • 스탭 배치 변경
  • 출연진 대기 / 순서 문제
  • 물판 정산 / 재고 문제
  • 식사 / 물품 수량 문제
  • 실무 조정이 필요한 상황
스탭 자체 처리 가능 단순 안내, 예약번호 조회, 입장 줄 정리, 팔찌 확인, QR/캡쳐 확인.

17:10특전회 / 물판 진행
18:40철수 / 원상복구 / 정산 / 분실물 확인
20:00완전 퇴관

입장 확인 루틴

조회 순서 예약번호 끝 4자리 → 입금자명 → 닉네임 → 루미ID 순서로 확인합니다.
기본 처리
  • 예매 정보 확인
  • 입금확인완료 여부 확인
  • 입장완료 처리
  • 이미 입장완료면 재입장 불가 안내
입금확인대기 상태 바로 입장 처리하지 말고 담당자 확인 후 진행해 주세요.
동명이인 대응 검색 결과가 여러 명이면 메아테, 닉네임, 예약번호를 함께 확인해 선택합니다.

양도 / 현장판매

양도 입장 양도는 가능하지만 개인 간 거래 및 분쟁은 루미벨이 책임지지 않습니다.
양도 확인 3종 원 예매자 성함 + 숫자 4자리 + 예매 확인 메일 또는 캡쳐.
현장 판매 20,000원 / 현금 또는 계좌이체만 가능 / 카드 불가.

물판 · 포인트 루틴

물판 확인
  • 구매 상품 / 수량 확인
  • 현금 또는 계좌이체 확인
  • 주문번호나 구매 내역 확인
  • 문제 발생 시 부총괄 호출
루미ID 포인트 팬의 루미ID 카드/번호를 확인하고 포인트 메뉴에서 적립 또는 조회합니다.
정산 주의 판매한 권 수량과 현금/계좌이체 매출이 맞는지 계속 확인해 주세요.

특전회 타이머

필수 멤버 옆 담당자는 반드시 소리 나는 타이머를 사용합니다.
알람 시 안내 “시간 다 되셨습니다, 이동 부탁드릴게요.”라고 정중하게 안내합니다.
대기열 줄이 밀리면 담당자에게 공유하고, 다음 팬을 빠르게 안내합니다.

대관처 기본 규정

식음료 반입 홀 내부 식음료 반입은 기본적으로 금지입니다. 생수만 허용되는 경우가 많으므로 당일 대관처 기준을 우선 적용합니다.
반입 제한
  • 주류 반입 금지
  • 커피 / 쿠키 등 식음료 반입 금지
  • 꽃다발 / 큰 선물은 제한될 수 있음
취식 구역 취식은 지정된 공간 또는 대기실에서만 가능합니다. 로비/홀 취식은 안내해 주세요.
쓰레기 음식물 쓰레기, 일반 쓰레기, 분리수거를 구분하고 퇴관 전 원상복구합니다.

팬 응대 멘트

입장 확인“예약번호 끝 4자리 알려주세요.”
기본 응대“확인 도와드릴게요.”
동명이인“동명이인이 있어 메아테도 확인 부탁드려요.”
대기 요청“잠시만 기다려주시면 담당자 확인 후 안내드릴게요.”
반입 금지“생수 외 식음료는 홀 반입이 어려워요.”
특전회 종료“시간 다 되셨습니다, 이동 부탁드릴게요.”

문제 상황 대응

클레임 바로 논쟁하지 말고 “담당자 확인 후 안내드리겠습니다.”라고 말한 뒤 총괄/부총괄을 호출합니다.
분실물 발견 즉시 총괄/부총괄에게 전달합니다. 스탭 개인 보관이나 개인 연락은 하지 않습니다.
응급 상황 주변 공간을 확보하고 총괄/대관처 직원을 호출합니다. 혼자 판단하지 않습니다.
촬영/녹음 공연별 공지를 우선 적용합니다. 금지 구역/금지 시간에는 즉시 안내하고, 팬 기기는 직접 만지지 않습니다.

퇴관 체크

  • 쓰레기 수거 및 분리수거 완료
  • 대기실 개인 물품 확인
  • 남은 굿즈 / 특전권 회수
  • 금고 / 현금 / 계좌 정산 확인
  • 분실물 확인 및 담당자 전달
  • 대관처 비품 원상복구
  • 전 스탭 퇴근 여부 확인
